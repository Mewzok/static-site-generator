# Tools I Use Every Day

These are the tools that have actually stuck around on my machine for more than a month.

## VS Code

Hard to argue with. The Python extension and integrated terminal cover most of what I need day to day.

## Windows Terminal

A massive improvement over the default command prompt. Tabs, better fonts, and actual color support make a real difference when you're staring at a terminal all day.

## Python + pathlib

For anything involving files, pathlib makes the code readable enough that I can come back to it six months later and still understand it.

## Git

Obvious, but worth saying. Even for solo projects, having a commit history has saved me more than once.
